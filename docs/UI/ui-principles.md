## 3. Navigation Architecture
* **The Global Drawer:** Acts as the "Identity Anchor." It must feel like a safe space.
* **Bottom Tabs:** Limit to 3-4 tabs maximum per module to prevent choice paralysis.
* **The "Identity Card":** The top of the drawer features the user's avatar and age, reinforcing the "Future Self" continuity (critical for combatting depressive apathy).
